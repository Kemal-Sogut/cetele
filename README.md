# Cetele

A React-based web application for tracking daily activities, prayers, and personal goals, featuring a dashboard and mentoring system.

## Features

- **Activity Tracking**: Record daily entries for various activities.
- **Prayer Tracking**: Specialized interface for tracking daily prayers.
- **Dashboard**: Personal dashboard to visualize progress and statistics.
- **Mentoring System**: Dedicated mentor dashboard for overseeing progress.
- **Firebase Integration**: Real-time database and authentication via Firebase.
- **Responsive Design**: Built with Tailwind CSS for mobile and desktop support.

## Tech Stack

- **Frontend**: React (v18), React Router
- **Styling**: Tailwind CSS, PostCSS
- **Backend/Database**: Firebase (Authentication & Cloud Firestore)
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js installed on your local machine
- A Firebase project set up

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kemal-Sogut/cetele.git
   cd cetele
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Firebase configuration variables (you will need to configure these based on your Firebase project setup).

4. Start the development server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the application for production.
- `npm run preview`: Previews the production build locally.
- `npm start`: Runs the Node server.
